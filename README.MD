# Расширение с отзывами на преподавателей

**Два режима работы!**
- Быстрый поиск по предмету или преподавателю в окне расширения.
- Встраивание на страницу преподавателя на сайте.

**Репутация**: можно голосовать за полезность отзыва и ставить рейтинг преподавателю, остальные увидят общую оценку!

**Кто вы?** Расширение само определит, кто вы, как только зайдёте на сайт. **Ваши данные не сохраняются**, определение нужно только для того, чтобы нельзя было накрутить карму или рейтинг повыше)

# Для желающих разрабатывать...

### Сборка расширения `build.js`

Установка
```shell
npm init -y
npm install --save-dev esbuild
npm install --save-dev cross-env
npm install --save-dev archiver
```

Сборка
```shell
node run build:prod
# или если api работает на localhost
node run build:dev
```

### Релиз `release.js`
```shell
node run release
```
- Версию нужно указывать в `manifest.json`.
- Для описания релиза будет использовано `release-notes.md`.
- Может сойти за глупенький ci/cd.

### Содержимое `src`

Изначально у нас два путя
1. `injector.js` производит встраивание на сайт блока отзывов и поиск jwt токена
2. `popup.html` запускает `popup.js` который обеспечивает работу окошечка

В `manifest.json` описаны основные характеристики расширения, необходимые браузеру.

В `style.css` хранятся топовые стили. **Ни в коем случае** не переназначайте общие стили элементов тут, используйте для этого `popup.html`, иначе переопределяться стили сайта.

В остальном разберётесь по ходу дела)
